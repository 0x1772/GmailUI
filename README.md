# GmailUI

This is a thin wrapper around the Gmail web interface built with
[Electron](https://electronjs.org).

## Features

- Native desktop notifications
- Dark sidebar theme
- Many small visual UI improvements

## Building

```
npm install
npm run pack-osx
cp -r Gmail-darwin-x64/Gmail.app /Applications
```
